A combined interactive dashboard built in Power BI and Tableau to analyze overall banking performance. It features key KPIs such as Total Funded Amount, Total Loan Count, Total Collection, and Total Interest Received. The dashboard includes visual charts for state-wise and branch-wise loan distribution, product-wise loan analysis, religion-wise and age-wise segmentation, and loan status tracking (active, fully paid, net-off, etc.). With dynamic slicers and drill-down charts, the dashboard helps identify top-performing regions, branches, and loan products for faster, data-driven decision-making.
